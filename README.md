# 🏨 Hotel Reservation Cancellation Prediction  
### Comparative Analysis of Machine Learning Models

## 📌 Overview
This project presents a comparative analysis of supervised machine learning models to predict hotel reservation cancellations. The goal is to identify the most effective model and understand the key factors influencing cancellation behavior.

Developed as part of the **Machine Learning for Aging and Longevity** course at **Technische Universität Berlin**.



---

## 🎯 Objectives
- Analyze patterns in hotel booking data  
- Perform exploratory data analysis (EDA)  
- Engineer meaningful features  
- Train and compare multiple machine learning models  
- Evaluate model performance using appropriate metrics  

---

## 📊 Dataset
The project uses the **Hotel Reservations Dataset**, which includes booking information such as:
- Lead time  
- Number of nights (week/weekend)  
- Average price per room  
- Customer and booking details  
- Cancellation status (target variable)  

---

## ⚙️ Project Workflow

### 1. Data Preparation
- Dataset loading  
- Train-test split with stratification  

### 2. Exploratory Data Analysis (EDA)
- Distribution analysis  
- Outlier detection  
- Feature relationships  

### 3. Feature Engineering
- Created new feature: `total_nights`  
- Encoding categorical variables  

### 4. Model Training
- Logistic Regression  
- Random Forest  
- Gradient Boosting  

### 5. Evaluation
- Accuracy  
- Precision / Recall  
- F1-score  
- Confusion Matrix
